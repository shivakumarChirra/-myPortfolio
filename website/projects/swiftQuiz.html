Swift Quiz App

About the App

Swift Quiz App is an interactive and educational application developed by Shivappz to test and enhance Swift programming knowledge. Designed with an intuitive user interface and built using Swift and UIKit, this app presents users with a series of true/false questions to help them reinforce core Swift concepts. This app was created as part of my learning journey to become an iOS developer, allowing me to explore key skills such as dynamic UI updates, data structuring, logic implementation, and user interaction. It highlights my dedication to building practical and educational tools from scratch while learning iOS development.

Technologies Used:

  • Dynamic UI and button interaction using @IBAction
  • Data handling through custom models and structs
  • Programmatic UI updates and feedback using UIKit
  • Logic implementation using control flow and conditional statements
  • Part of the Shivappz suite of apps — a collection showcasing my journey and projects as an iOS developer.
